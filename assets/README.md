# Assets Folder

## Required Files

Place the following files in this folder:

| File | Description | Notes |
|------|-------------|-------|
| `music.mp3` | Ambient romantic background music | Keep under 5MB for fast loading |
| `petals.png` | (Optional) Petal texture | Not required - CSS generates petals |
| `tulip-texture.png` | (Optional) Tulip texture overlay | Not required - CSS generates tulips |
| `background.jpg` | (Optional) Fallback background | Not required - CSS generates sky |

## Recommended Free Ambient Music Sources

1. **Pixabay Music** - https://pixabay.com/music/
   - Search: "romantic ambient" or "peaceful nature"
   - License: Free for commercial use, no attribution required

2. **Free Music Archive** - https://freemusicarchive.org/
   - Search: "ambient romantic" or "sunset"
   - Check individual licenses

3. **Mixkit** - https://mixkit.co/free-stock-music/
   - Search: "romantic" or "emotional piano"
   - License: Free to use

4. **Bensound** - https://www.bensound.com/
   - Search: "romantic" or "love"
   - Free with attribution (or paid for no attribution)

5. **YouTube Audio Library** - https://studio.youtube.com/channel/audio
   - Filter by mood: "Romantic" or "Calm"
   - License: Free to use

## Tips
- Choose music that is soft, ambient, and loop-friendly
- Piano + strings combinations work beautifully
- Keep file size reasonable (3-8MB for a 3-minute loop)
- Name the file exactly `music.mp3`
